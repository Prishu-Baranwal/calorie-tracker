# AI-Based Daily Calorie & Protein Tracker

## Overview
This project uses Deep Learning and Machine Learning to:
- Detect food from images
- Estimate calories and protein
- Track daily nutrition
- Predict weekly health risks

## Tech Stack
- Python
- TensorFlow/Keras
- OpenCV
- Scikit-learn
- Streamlit
- Pandas
- NumPy
